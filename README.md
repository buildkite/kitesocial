## Welcome

Welcome to Kitesocial, a Twitter clone!

## Development setup

To run Kitesocial, you will need:

- Ruby 3.2.2
- Rails 6.0.3
- Node 16.12.0
- Yarn 1.19.1
- Chrome or Chromium

Once you've cloned the repo, run:

```
  bin/setup
```

Then you'll be able to spin up your server and run your specs!
